Tagging Tasks

A tag is a simple word that begins with the @ sign.

When you type a word beginning with @, it displays as highlighted yellow, which is the indicator that GTG considers the word to be a tag.

GTG Tag

Tags are useful to sort your tasks. You can enable the Sidebar from the Main Menu on the Headerbar. This displays all of the tags that you have created. When a tag is selected, all tasks that are associated with that tag are displayed. There is no limit to the number of tags that a task can have.
