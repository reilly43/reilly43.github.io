Enable/Disable Sync Service

To enable or disable Sync Services in the GTG main menu go to Edit ▸ Setup Sync Services. Press the button Add and choose the desired sync service from the drop down menu on the right.

Enable Sync Service

For each sync service that you wish to enable you will have to set certain parameters, provide the user name and password if necessary and press the Enable syncing button.

To disable a sync service press the Disable syncing button.
